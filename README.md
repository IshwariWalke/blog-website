# blog-website

WordPress.com Clone

Overview
This project is a static clone of the WordPress.com homepage created using HTML and CSS. It replicates the layout and design of WordPress.com, but does not include dynamic functionality or backend features.

Features
Static HTML/CSS layout mimicking the design of WordPress.com.
Basic navigation and page structure.

Features Not Included
Backend functionality (e.g., user authentication, content management).
JavaScript-based interactivity or dynamic content.

1. Clone the Repository:
   git clone <https://github.com/IshwariWalke/blog-website.git>

2. Navigate to the Project Directory:
   cd <project-directory>

3. Open the index.html file in a web browser
